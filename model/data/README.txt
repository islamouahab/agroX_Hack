## Files associated with "Correlates of hybridization in plants"

-family.tre - family-level phylogeny used in analyses
-genus.tre - genus-level phylogeny used in analyses
-family_data.csv - family-level data used in analyses
-genus_data.sv - genus-level data used in analyses
